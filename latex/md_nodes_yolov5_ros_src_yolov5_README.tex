Training 

The commands below reproduce Y\+O\+L\+Ov5 \href{https://github.com/ultralytics/yolov5/blob/master/data/scripts/get_coco.sh}{\texttt{ C\+O\+CO}} results. \href{https://github.com/ultralytics/yolov5/tree/master/models}{\texttt{ Models}} and \href{https://github.com/ultralytics/yolov5/tree/master/data}{\texttt{ datasets}} download automatically from the latest Y\+O\+L\+Ov5 \href{https://github.com/ultralytics/yolov5/releases}{\texttt{ release}}. Training times for Y\+O\+L\+Ov5n/s/m/l/x are 1/2/4/6/8 days on a V100 G\+PU (\href{https://github.com/ultralytics/yolov5/issues/475}{\texttt{ Multi-\/\+G\+PU}} times faster). Use the largest {\ttfamily -\/-\/batch-\/size} possible, or pass {\ttfamily -\/-\/batch-\/size -\/1} for Y\+O\+L\+Ov5 \href{https://github.com/ultralytics/yolov5/pull/5092}{\texttt{ Auto\+Batch}}. Batch sizes shown for V100-\/16\+GB.
